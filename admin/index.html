가이드 라인

  1. 가입 상품명 입력 및 샘플양식에 맞춰 스크립트 파일 업로드
  2. TTS 유형에서 남성 또는 여성 목소리 선택 후 음성 파일 생성 클릭
  3. 오른편 화면에 약관 스크립트 LIST 생성시 약관 번호 별 음성 파일 각각 확인
  4. 문제 없을 시 등록 클릭
  5. 잘못 된 음성 또는 오류 발생시 삭제 클릭
  6. 파일 수정 후 재 업로드 및 음성 파일 다시 생성
